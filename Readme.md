## Run Locally

Clone the project

```bash
  git clone https://github.com/vahidrezazadeh/sep-gateway-django.git
```

Go to the project directory

```bash
  cd sep-gateway-django
```

Install dependencies

```bash
  pip install -r Pipfile
```

Start the server

```bash
  python manage.py runserver
```
